Feature 5: STIX/TAXII Feed Parsing

What It Does:
    - Fetches threat intelligence feeds from free MISP/CIRCL and other TAXII sources
    - Parses STIX format feeds into n8n-compatible JSON format
    - Normalizes IOCs to match the same structure as other pipeline features
    - Sends parsed threat intel to n8n Cloud webhook: https://sipiv63984.app.n8n.cloud/webhook-test/stix-taxii

Free STIX/TAXII Sources:
    - MISP Project feeds (misp-project.org)
    - CIRCL feeds (circl.lu)
    - US-CERT feeds (cisa.gov)
    - OpenCTI community feeds
    - FBI InfraGard (public feeds)

Input Data:
    - STIX 2.1 JSON feeds
    - TAXII collections
    - XML/JSON threat intel feeds
    - RSS/Atom feeds with indicators

Output:
    - Normalized IOC objects (IPs, domains, hashes, URLs)
    - Threat actor information
    - Campaign and malware family data
    - Attack patterns and TTPs
    - Structured JSON for n8n processing

Key Features:
    - Support for STIX 2.1 format
    - Rate limiting for feed sources
    - Deduplication of indicators
    - IOC validation and enrichment
    - Metadata extraction (confidence scores, tags, etc.)
