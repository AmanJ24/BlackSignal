# Reputation Scoring

This feature evaluates and assigns reputation scores to vendors based on feedback mechanisms, BTC transaction paths, and frequency of profile mentions within dark web marketplaces.

## Input
- Scraped vendor data (JSON)

## Output
- Reputation Score

## Usage Example
```bash
cd PROJECT_DARK_WEB/FEAT_13_REPUTATION_SCORING
python3 reputation_scorer.py
```

## Dependencies
- Python libraries: `json`, `requests`
- External services: None (mock data used)
