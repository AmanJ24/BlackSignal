# 🕵️ Named Entity Recognition (NER) - Feature 7

**Purpose**: Detects threat actor handles, organization names, and group affiliations from dark web content.

## 🧰 Tools Required
- `spaCy` (≥3.4.0) with `en_core_web_sm` model
- `requests` (≥2.28.0)
- `flask` (≥2.2.0) - Optional for API setup
- `python-dotenv` (≥0.19.0)

## 📁 File Structure
```
NER_FEAT_7/
├── ner_feature.py              # Main NER processing script
├── sample_dark_web_data.json   # Input data from other features
├── requirements.txt            # Python dependencies
└── instructions.txt           # This file
```

## 🚀 Setup Instructions

### 1. Create Virtual Environment
```bash
python3 -m venv ../venv_ner
source ../venv_ner/bin/activate
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
python -m spacy download en_core_web_sm
```

### 3. Prepare Input Data
Ensure `sample_dark_web_data.json` contains data from other features:
```json
[
    {
        "source": "marketplace_scraper",
        "timestamp": "2025-07-31T10:30:00Z",
        "data": "Vendor CryptoKing77 from DarkBazaar Network..."
    }
]
```

### 4. Run NER Processing
```bash
source ../venv_ner/bin/activate
python ner_feature.py
```

## 🎯 What It Detects

- **PERSON**: Threat actor handles, individual names
- **ORG**: Criminal organizations, marketplaces, groups
- **NORP**: Nationalities, political/religious groups

## 📊 Output Format

```json
{
  "timestamp": "2025-07-31T11:02:52.952266",
  "feature": "Named Entity Recognition (NER)",
  "input_text": "Vendor CryptoKing77 from DarkBazaar Network...",
  "entities_found": 3,
  "entities": {
    "PERSON": ["Anonymous Collective"],
    "ORG": ["DarkBazaar Network"],
    "NORP": ["Russian"]
  },
  "raw_entities": [["DarkBazaar Network", "ORG"]],
  "source": "marketplace_scraper",
  "source_timestamp": "2025-07-31T10:30:00Z"
}
```

## 🔗 Integration with OSINT Pipeline

1. **Input**: Receives text data from Features 1-6 via `sample_dark_web_data.json`
2. **Processing**: Extracts named entities using spaCy NLP
3. **Output**: Structured JSON ready for n8n webhook integration
4. **Next Steps**: Results feed into Features 8-15 for correlation and analysis

## ⚙️ Functions

- `extract_entities(text)`: Core NER processing with spaCy
- `process_text(text)`: Formats results for webhook integration
- `load_sample_data(json_file)`: Loads input data with error handling

## 🔧 Customization

**To add more entity types**:
```python
named_entities = [(ent.text, ent.label_) for ent in doc.ents 
                 if ent.label_ in ["PERSON", "ORG", "NORP", "GPE", "MONEY"]]
```

**To change input file**:
```python
sample_data_list = load_sample_data("custom_input.json")
```
